I'm Anthony, a Master's student at Rice studying Computational and Applied Mathematics. I have a strong CS background and have built production ML backends serving thousands of users, led full development lifecycles from architecture to deployment, and love diving deep into systems: from GPU simulation research to running my own homelab.


## Contributions

In addition to the many repos I've created, I'm also a maintainer of the following:

- The plugin system of [Zoraxy](https://github.com/tobychui/zoraxy): A general purpose HTTP reverse proxy and forwarding tool.
- [tsync](https://github.com/Wulf/tsync): Synchronize rust and typescript types!
- [create-rust-app](https://github.com/Wulf/create-rust-app): Set up a modern rust+react web app by running one command. 

I've also made (what I would consider substaintial) contributions to:

- [bliss-rs](https://github.com/Polochon-street/bliss-rs): I upstreamed the symphonia based decoder from [mecomp-analysis](https://crates.io/crates/mecomp-analysis)
- [pywal16](https://github.com/eylles/pywal16): I wrote a top-down recursive-descent parser to replace buggy templating logic.

For a full list, see [this github search](https://github.com/search?q=is%3Apr+-user%3AAnthonyMichaelTDM+author%3AAnthonyMichaelTDM++is%3Amerged+&type=pullrequests&ref=advsearch&s=comments&o=desc)
